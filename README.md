# Lang Code

## Overview

Lang Code is an AI-powered, command-line interface (CLI) assistant designed to enhance and automate software development workflows directly from your terminal. It acts as a sophisticated coding agent, capable of understanding project context, generating code, refactoring existing files, debugging, and interacting with the file system—all while providing a rich Text User Interface (TUI) experience.

### Features
*   **Context-Aware Coding:** Understands the structure and content of your current project directory to provide relevant suggestions.
*   **Agentic Workflow:** Implements multi-step reasoning and execution via specialized agents for complex tasks (e.g., "Implement feature X" or "Refactor component Y").
*   **File System Interaction:** Provides tools to read, write, and manage files within the project scope.
*   **Rich TUI Experience:** Offers an interactive, color-coded terminal interface for a seamless user experience.

### Getting Started
* **Installation:** Ensure you have Python and uv installed.
* **Running:** To start the application, run the following command in your terminal:

```sh
uv run code
```
